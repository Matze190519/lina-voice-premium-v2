## 1. Design System (Updated)

### Color Palette
*   **Primary Background:** Deep Navy / Black (`#0B0E18`) - Matches website `bg-deep-space`
*   **Primary Accent:** Neon Purple (`#C49BFF`) - Matches website `text-neon-purple`
*   **Secondary Accent:** Vibrant Cyan (`#00E5FF`) - Matches website `text-cyan-400`
*   **Text:** White / Light Grey - High contrast for readability

### Visual Style
*   **Glassmorphism:** Frosted glass cards with thin, glowing borders.
*   **Lighting:** Cinematic rim lighting in purple and cyan.
*   **Imagery:** High-end B2B environments (offices, workshops) blended with futuristic UI overlays.
